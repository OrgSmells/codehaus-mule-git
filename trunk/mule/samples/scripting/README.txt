+-------------------+
| Scripting Example |
+-------------------+
A couple of simple examples that demonstrate Mule's JSR-223 Scripting support.

For more information, refer to http://www.muleumo.org/Examples

If you haven't yet added Mule to your executable path, please follow the instructions
in INSTALL.txt first.  To run this example:

Linux / Unix
------------
mule -main org.mule.samples.scripting.BinaryHttpExample
mule -main org.mule.samples.scripting.TextFileExample

Windows
-------
mule.bat -main org.mule.samples.scripting.BinaryHttpExample
mule.bat -main org.mule.samples.scripting.TextFileExample
