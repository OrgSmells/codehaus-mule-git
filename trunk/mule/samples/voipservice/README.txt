+----------------------+
| VoIP Service Example |
+----------------------+
This example is described in the feature Java.net article
"Service Provisioning Through ESB".

For more information, refer to http://www.muleumo.org/Examples

If you haven't yet added Mule to your executable path, please follow the instructions
in INSTALL.txt first.  To run this example:

Linux / Unix
------------
mule -main org.mule.samples.voipservice.client.VoipConsumer

Windows
-------
mule.bat -main org.mule.samples.voipservice.client.VoipConsumer
